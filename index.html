Disco at the Dungeon

Disco at the Dungeon is an app that helps the user keep track of their D&D character stats.

This is a MERN stack single-page application that was created using GraphQL, MongoDB,Express.js, Node.js, JWT, Apollo, React, CSS, and Bulma.

Date Night at Home

Date Night at Home is a simple site that helps you plan the perfect date night for you and your partner. It allows you to search for a dish you'd like to create and we'll offer a wine match as well as provide you with a recipe to cook.

This app was created using Git, HTML, CSS, Bulma, Tasty API and Spoonacular API.

Friendless in Seattle

Friendless in Seattle is an app designed to help the user meet new friends locally or around the world that share your common interests.

This app was created using Git, HTML, CSS, Bulma, JavaScript, JQuery, Handlebars, Node.js, Express.js, MySQL, Sequelize, and an API that randomly generates an Avatar based on the username.
